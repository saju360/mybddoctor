# Bangladesh Healthcare Platform

Monorepo for a dynamic healthcare ecosystem focused on Bangladesh.

## Projects

- `android-app` - Java Android client (user-facing app)
- `backend-api` - Spring Boot REST API with MySQL
- `admin-web` - React + Vite admin panel
- `docs` - architecture, API, and rollout docs

## Quick Start

1. Start MySQL with Docker compose.
2. Run Spring Boot API from `backend-api`.
3. Run Admin panel from `admin-web`.
4. Open `android-app` in Android Studio and build.
